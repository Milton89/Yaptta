Installation Notes
==================

- To install Yaptta, copy the Yaptta-signed.apk file to your device and access the
  APK file from a file manager such as ES File Explorer



Usage Notes
===========

- After setting your device name for the first time, please close the application
  (press back until it closes) and restart it. There is a known issue where the
  device name does not get properly propagated to internal data structures unless
  the application is reset.

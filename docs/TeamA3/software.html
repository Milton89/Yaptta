Yaptta: Yet Another Push-To-Talk Application

Software

The software listing provided below is a snapshot of the main development platform used for this project. Other configurations are certainly possible; as long as compatible versions are used, there should be no issues in working with the software. No custom builds of any tools were used; everything was "off the shelf."

Main Development Platform

  • Windows 8
  • git version 1.8.1.msysgit.1 (for SCM)
  • Eclipse Juno SR2 (Java IDE)
    • Java EE version recommended
    • with Android Development Toolkit
    • and Android SDK installed separately (select "Use an Existing IDE")
    • with Java VM: 
      java version "1.7.0_09"
Java(TM) SE Runtime Environment (build 1.7.0_09-b05)
Java HotSpot(TM) 64-Bit Server VM (build 23.5-b02, mixed mode)
